# Memory Policy Workspace

This is the editable surface for the autonomous research loop.

Artifacts:

- `paper/abstract.md`: concise paper pitch
- `paper/outline.md`: full narrative structure
- `design/memory-loop.md`: system architecture and policy details
- `experiments/plan.md`: experiment strategy and rollout phases
- `experiments/eval-matrix.md`: baselines, metrics, ablations, and outcome table

The controller accepts only files inside this directory.
